# Checklist danh gia cong cu

## Bat buoc

- [ ] Cõ mã nguồn Python, chia module/provider.
- [ ] Có giao diện trực quan hoặc CLI để người dùng chạy lại.
- [ ] Tièm kiếm theo bbox, thời gian, bộ dữ liệu, cloud cover.
- [ ] Tải dữ liệu hoặc asset từ EarthSearch.
- [ ] Tải dữ liệu hoặc asset từ Planetary Computer.
- [ ] Kết nối Copernicus/ESA bằng tải khoản và product.
- [ ] Kết nối USGS EarthExplorer/M2M bằng application token và tải product.
- [ ] Kết nối Google Earth Engine và export vùng nhỏ.
- [ ] Lưu metadata kết quả thành CSV/JSON.
- [ ] Không hardcode username/password/token trong code.
- [ ] Có README huong dẫn cài đặt, chạy, demo và đẩy Git.
- [ ] Có `.gitignore` chặn file lớn và file bị mất.
- [ ] Có ít nhất vài test nhỏ cho phần lõi.

## Diem can cai thien

- [ ] Xư lý lỗi mạng, timeout, token hết hạn.
- [ ] Có log tiến trình tải file.
- [ ] Cho phép batch download nhiều cảnh.
- [ ] Cho phép chọn bands cần tải.
- [ ] Có ảnh chụp màn hình demo giao diện.
- [ ] Có notebook minh họa quy trình khoa học.
- [ ] Có bảng so sánh ưu/nhược điểm các nguồn.
- [ ] Có mô tả giới hạn API và yeu cau tai khoan.
